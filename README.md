# 📘 Curso: Engenharia de Software (2026)

## 🧾 Descrição
Curso completo que aborda os principais conceitos, práticas e ferramentas da Engenharia de Software. Você aprenderá desde o desenvolvimento até a implementação de sistemas robustos, incluindo qualidade, testes, manutenção e gestão de projetos.

## 🚀 O que você aprenderá
- Desenvolvimento e Engenharia de Software  
- Testes e Qualidade de Software  
- Gestão de Projetos e Riscos  
- SCRUM e UX  
- Requisitos e Arquitetura de Software  
- Segurança da Informação  
- Estrutura de Software  
- Dados e Análise de Dados  
- Profissão de Desenvolvedor e Engenheiro de Software  

## 📦 Recursos
- Materiais das aulas disponíveis para download  
- Conteúdo didático, simples e direto  

## 📌 Pré-requisitos
- Apenas vontade de aprender  

## 🎯 Público-alvo
- Iniciantes e profissionais que desejam evoluir na área de tecnologia  

## 💡 Objetivo
Capacitar o aluno para enfrentar desafios reais do desenvolvimento de software e se destacar no mercado de tecnologia.
